Saving a File

If you have created a new file and have not saved it yet, it will have the name Untitled.dva. Any save command will automatically take you to the Save As... dialog box.

To save a file which you have previously saved:

Press Ctrl+S, or click File, Save.

To save a file with a different name or format:

Click File, Save As....
Choose the format (.dva or .mva) and enter a new name in the text box
Click Save.
