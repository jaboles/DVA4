Compiling a File

DVA can compile both .dva and .mva files to an MS-DOS Batch file. This is useful if playing through Windows is too slow or low in quality.

To compile a file:

From a Windows Explorer window, right-click the file and then click Compile.
Browse for the directory where you want to place the compiled file.

or

Open the file you wish to compile in DVA.
Press F6, or click File, Compile.
Browse for the directory where you want to place the compiled file.
