Opening a File

Opening a file places it in the editing area. From there you can edit, play, or compile it. DVA can open both .dva and .mva files.

To open a file:

From a Windows Explorer window, simply right-click the file and click Open.

or

Start DVA.
Press Ctrl+O, or click File, Open.
Browse for the file, select it, then click Open.

Note: If you have specified in the Options, File Associations dialog, you can also double-click the file to open it.
