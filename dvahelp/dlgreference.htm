Dialogs in DVA

This section explains the dialogs you will encounter in DVA.

The Find dialog allows you to search the editing area for a text string.

The Not Found dialog lets you know when there are no more occurences of your text string

The Preferences dialog allows you to modify DVA program options.

The Properties dialog allows you to view and edit information about the current file.

The Replace dialog allows you to replace all instances of one text string with another text string.

The Save Changes dialog reminds you to save changes when you load a new file, create a new file, or exit DVA.
