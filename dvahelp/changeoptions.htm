Changing Program Options

This help topic documents how to change options/preferences for DVA.

Sounds Directory
The sounds directory setting lets you decide the location where sounds which are used by DVA, are searched for. If you use different kinds of announcement, for example different voices, you should store them in different folders to prevent overlapping of filenames. Then, to change the style of announcement, simlply switch the sounds directory.

To change the Sounds Directory:

Click Options, Sounds Directory....
Using the list box, navigate to a new sounds directory.
When you have selected a sounds directory, click Close to save the settings and close the dialog box.

Source Directory
The source directory setting lets you control where DVA and MVA files are searched for by default. Whenever you save, open, or create a DVA or an MVA file, this will be the suggested directory.

To change the Source Directory:

Click Options, Source Directory....
Using the list box, naviagate to the new default source directory.
When you have selected a source directory, click Close to save the settings and close the dialog box.

Default Shell Action
The Default Shell Action dialog box allows you to control what will occur if you double click a file in Windows Explorer.

To change Shell Action options:

Click Options, Default Shell Action....
Select, using the radio buttons, whether you want a file to be opened, played or compiled when it is double-clicked in Explorer.
Click OK to save changes, or Cancel to leave the settings as they were.

Print Output
This menu item displays a dialog box allowing you to specify a file or device to print to when the File | Print menu item is chosen.

To specify a device to print to:

Click Options, Set Print Output....
Enter the name of the file or device in the edit box.
Click OK to change the setting, or Cancel to leave it as it was.
