Menus in DVA

The menus contain commands for getting help and working with DVA and MVA files and their text.

The File menu contains commands for working with particular files.

The Edit menu has commands for manipulating text in the editing area.

The View menu has commands for changing the appearance of the window.

The Search menu has commands for searching for and replacing text within the editing area.

The Options menu contains commands for changing DVA program default options.

The Help menu takes you to instant local and online help.
