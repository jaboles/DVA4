Keyboard Commands

These are all the keyboard shortcut commands you can use to access dialogs.
    File Menu
  • Ctrl+N: Creates a new file.
  • Ctrl+O: Opens an existing file.
  • F3: Plays the currently open file.
  • F4: Displays a dialog to compile the currently open file.
  • Ctrl+S: Saves the currently open file. If the file is untitled, you are prompted for a name and location.
  • Ctrl+P: Prints the file to the device or file specified in Options | Set Print Output...
  • F5: Refreshes the file list.
  • Alt+F4: Closes DVA. You are prompted to save changes.

    Edit Menu

  • Ctrl+X: Deletes the selected text and places it on the clipboard.
  • Ctrl+C: Places selected text on the clipboard without deleting it.
  • Ctrl+V: Inserts text on the clipboard into the editing area at the insertion point.
  • Del: Deletes selected text.
  • Ctrl+A: Selects all text in the text editing area.

    Search Menu

  • Ctrl+F: Displays a Find dialog allowing you to search for a text string.
  • F3: Find Next.
  • Ctrl+H: Displays Replace dialog so you can replace all instances of particular text with different text.

    Help Menu

  • F1: Displays Help contents and topics.
  • Shift+F1: Displays Help index.
