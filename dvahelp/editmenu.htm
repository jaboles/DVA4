Edit Menu

The Edit menu (Alt+E) contains commands for working with text in the editing area.

  • Cut (shortcut Ctrl+X) Deletes selected text and placed it in the clipboard.
  • Copy (shortcut Ctrl+C) Places selected text in the clipboard without deleting it.
  • Paste (shortcut Ctrl+V) Inserts any text in the clipboard at the insertion point.
  • Select All (shortcut Ctrl+A) Selects all text in the editing
  • Clear (shortcut Del) deletes the selected text. area.
