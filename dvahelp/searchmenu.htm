Search Menu

The search menu contains commands for finding and replacing text within the editing area:

  • Find... (shortcut Ctrl+F) displays a dialog box where you can search for a text string within the currently open DVA or MVA file.
  • Find Next (shortcut F3) re-executes the last find, beginning from where the last string was found. Note that if you select Find... again, any subsequent searches will take place from the beginning of the open file.
  • Replace... (shortcut Ctrl+H) searches for all instances of a particular item of text and replaces it with text you specify. If you select text within the editing area before you select Replace, this text will automatically be entered into the Replace edit box.
